# research/

Running notebook for the reverse-engineering work. Not polished docs —
this is where hypotheses, diff logs, dead ends, and "huh, that's weird"
observations live so we don't lose them across sessions.

- `format_notes.md` — the master running document. Append-mostly. Each
  finding gets a date, the sample(s) it came from, and the hypothesis +
  evidence.
- `diffs/` — saved binary diffs between sample pairs (created as we go).
- `hexdumps/` — annotated hex dumps for specific findings.
